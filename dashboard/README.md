# Nordic Retail Delivery Performance Dashboard

One-page Power BI dashboard built on top of the datasets generated in the analysis notebook.  
The dashboard focuses on delivery reliability and revenue trends across Danish regions (Copenhagen, Aarhus, Odense).

**Data source:** CSV files generated via Python + SQL and stored in `/data`.

## Dashboard Overview

The interactive Power BI dashboard highlights:
- Monthly on-time delivery performance
- Delivery reliability by region
- Revenue distribution across regions

The goal is to provide a clear, at-a-glance view of operational performance and regional differences.
